# PF2 Unlimited Metal During Setup Sourcemod plugin
Sourcemod plugin for [Pre-Fortress 2](https://prefortress.com) that allows the Engineer to have unlimited metal during setup. 

## Usage:
Requires Sourcemod 1.11+. Will automatically allow the Engineer to have "unlimited" metal during "waiting for players" and "setup" time. If there is no setup time in the map, the Engineer will operate as normal.
